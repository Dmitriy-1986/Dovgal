Как необходимо верстать сайты. Ключевые моменты, которые считаю важными в этой работе.

п.1. Работа с Front-end разработкой сайта.

Открыв редактор кода и дизайн макет сайта, на рабочем столе компьютера создаётся папка, которая открывается в редакторе кода, в которой будут храниться директории будущего проекта.

Создавая главную страницу сайта - index.html, по порядку создается папка assets, в которой будут храниться папка CSS стилей и файл style.css, папка JavaScript с файлом scripts.js, а также папка images, в которой будут храниться картинки, а также папку fonts в которой будут храниться файл шрифта.

п. 1.1. Создание разметки веб страницы. 

   
<!DOCTYPE html >
<html lang="ru" >
  <head >
    
    <meta charset="utf-8" >
    <meta name="viewport" content="width=device-width, initial-scale=1" >

    
    <link href="./assets/CSS/style.css" rel="stylesheet" >

    <title >Заголовок </title >
 </head >
  <body >
       

     
    <script src="./assets/JavaScript/scripts.js">

Теперь можно приступать к созданию веб страницы согласно дизайну макета сайта.

п.п.1.2. Работа с файлами стилей.

В файле style.css подключаем шрифт, создаём в директории assets папку bloks в которой создаём файлы стилей основных блоков. 

@import url('../fonts/roboto.woff2');
@import url('./assets/core.css');
@import url('./assets/reset.css');
@import url('./assets/bloks/header.css');
@import url('./assets/bloks/main.css');
@import url('./assets/bloks/section.css');
@import url('./assets/bloks/button.css');
@import url('./assets/bloks/footer.css');

п.п.1.3. Файл scripts.js.

В файле хранятся скрипты, которые будут использоваться на всех страницах сайта. Скрипты которые не распространяются на все страницы сайта, создаются и подключаться отдельно.

п.2. Работа с Back-end разработкой сайта.

В создании back-end использую ООП - объектно ориентированное программирование с архитектурой MVC - models, views, controllers, что отвечает за клиентскую сторону сайта (шаблон), соединение с базой данных и функционал сайта. Также в проекте может понадобиться подключение пакетов PHP, потому необходимо создать файл JSON в котором будет подключение пакетов и в которой будет установлен соответствующий пакет.

п.2.1. Создание структуры базы данных MySQL в phpMyAdmin.

Создаётся новая база данных, указываю ее имя, выбираю кодировку utf-8 general-ci и приступаю к созданию таблицы базы данных и ее полей. С которой делается соединение в соответствующей директории.


Так я представляю создание веб - сайта и веб - приложения с использованием языка разметки HTML, языка стилей CSS, языка программирования JavaScript, языка программирования PHP, системы управления базами данных MySQL.
